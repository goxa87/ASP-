Это мой сайт-визитка

технология ASP.Net Core 3.1
Паттерн MVC
БД MSSQL
Front-end: чистый css и немного JQuery.

Реализованы:
*авторизация на основе ролей.
*главная страница(обо мне).
*страница работ с возможностью скачивания , комментирования и оставления лайков.
*Онлайн чат на основе SignalR.
*Контакты с формой обратной связи.
*Блог с возможностью оставления комментариев и лайки.
*Административная часть с возможностью управлять ролями пользователей, и удалением пользователей. 
Просмотр сообщений добавление работ и постов в блог.

Реализована зацита от SSX, Open redirect, SQL-injection, CSRF, Click-Jacking атак.
